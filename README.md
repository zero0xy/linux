# linux

### `mkdir`

Apa itu `mkdir`? `mkdir` adalah `make directory` kegunaannya adalah untuk membuat suatu folder

```
Contoh perintah dasar dari [ mkdir ] adalah sbb :

$ mkdir  < *folder1* >

*[* Berfungsi untuk membuat _folder baru_ dengan nama ( *folder1* ) dan usahakan nama tidak menggunakan _spasi_ *]*

$ mkdir < *folder1 folder2 folder3* > 

*[* Berfungsi untuk membuat _tiga folder_ sekaligus dengan menggunakan nama ( *folder1, folder2, folder3* ) *]*

$ mkdir < *.foldertersembunyi* >

*[* Berfungsi untuk membuat _folder tersembunyi_ dengan memberikan tanda ( *.* ) di awal nama folder dengan _nama folder_ ( *foldertersembunyi* ) dan untuk mengecek _folder tersembunyi_ tsb kita cukup mengetikkan _ls -a_ pada terminal *]*

$ mkdir < *folder\ saya* >

*[* Berfungsi untuk membuat _folder_ dengan _spasi_ juga dengan memberikan tanda ( *\* ) bisa juga dengan memberikan tanda ( *" "* ) dan tanda ( *' '* ) sehingga menghasilkan _nama folder_ ( *folder saya* ) *]*

$ mkdir < *directory1/folder1* >

*[* Berfungsi untuk membuat _folder_ di dalam ( *directory1* ) dengan nama _folder_ ( *folder1* ) *]*

$ mkdir < *{'folder satu', 'folder dua'}* >

*[* Berfungsi untuk membuat 2 folder sekaligus menggunakan spasi dan tanda ( *{' ', ' '}* ) ]*

$ mkdir < *-v foldersaya* >

*[* Berfungsi untuk menampilkan _info_ dari  ( *foldersaya* ) *]*

$ mkdir < *directory1/* >

*[* Berfungsi untuk menampilkan _info folder_ dari directory ( *directory1* ) *]*

$ mkdir < *-p foldersaya/folder1* >

*[* Berfungsi untuk membuat _directory_ dan _sub directory_ sekaligus dengan menggunakan tanda ( *-p* ) yang dimana _directory nya_ adalah ( *foldersaya* ) _dan sub directory nya_ adalah ( *folder1* ) *]*
```
